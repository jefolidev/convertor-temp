## Como rodar o projeto

```bash
# Clone este repositório
$ git clone https://github.com/jefolidev/convertor-temp.git

# Acesse a pasta do proejeto no terminal
# cd convertor-temp

# Instale todas dependências
$ npm install

# Execute a aplicação
```
